# ✨ Getting Started with Discord Bots

**Robo.js** makes building bots with **[Discord.js](https://discord.js.org)** as easy as placing files in the right place.

Create a **[Slash Command](./commands)** by making a file in the `/src/commands` directory. The name of the file becomes the name of the command. **[Context Commands](./context-menu)** and **[Events](./events)**? Just as easy!

This is more than just a command handler - it's a full-fledged **[Framework](/robojs/overview)** and **[Ecosystem](/plugins/directory)**.

:::tip Robo.js does _not_ replace Discord.js!

Instead, it simplifies development by managing your project. You still have all of **Discord.js**!

**Got an existing bot?** Check out the **[Migration Guide](./migrate)** for how to use existing code with **Robo.js**.

:::

## Quick Start

Ready to start your adventure? Try **[Create Robo](../cli/create-robo)** in your terminal:

<Terminal create>{'<projectName>'}</Terminal>

This will help spawn a new **Robo.js** project as a **Discord Bot**. We call these **Robos**.

## Plugin Power Ups

**[Robo Plugins](../plugins/overview)** are special. They let you add features to your **Robo** with one command.

<Terminal execute>robo add @robojs/ai @robojs/moderation</Terminal>

Plugins integrate seamlessly thanks to the **[Robo File Structure](../robojs/files)**. What's more, anyone can **[create a plugin](../plugins/create)**.
